# category_b

Приложение для просмотра случайных анекдотов с возможностью добавления в избранное и редактирования.
📱 Функционал
	•	🎲 Генерация случайных анекдотов
	•	💎 Избранное с локальным хранилищем (Hive)
	•	✏️ Редактирование анекдотов
	•	🎨 Светлая/темная тема (SharedPreferences)
	•	📱 Адаптивный UI
	•	🚀 AutoRoute навигация
